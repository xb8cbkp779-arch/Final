11 Virtues of Rational Character

Critical Thinking

  • real-time monitoring of own thinking
  • reasoning that holds up to challenge and stays consistent
  • reduce bias via monitoring of own thinking
  • contradiction-free conclusions

Sense-Making

  • single contradiction-free narrative
  • metacognition treated as evidence
  • assimilate allocentric perspectives
  • decisions run on tested mental models

Self-Control

  • inhibit impulsive actions
  • delay gratification as default exchange
  • goal pursuit without hedonic reward
  • maintain selective attention despite noise

Fairness

  • equity = contribution Â± need
  • engage in relational repair
  • uphold effort reciprocity
  • enforce rules with universality

Impartiality

  • anchor judgments in empirical facts
  • logic survives name-blank test
  • debiasing: pre-mortem / steel-man / outside view
  • intellectual humility rises as data falls

Strength & Courage

  • endure distress tolerance
  • confront phobias via exposure
  • stress inoculation via graduated load
  • anti-fragility gained from controlled stressors

Autonomy

  • choices guided by eudaimonic values
  • integrity overrides conformity pressure
  • exercise moral agency
  • harm never offloaded for gain

Self-Confidence

  • self-efficacy built on verifiable wins
  • successful performances fused into identity
  • observed successes converted into personal performance trials
  • setbacks read as skill deficits only

Persistence

  • perseverance after affect flatlines
  • obstacles treated as puzzles
  • momentum via daily compound actions
  • self-monitoring of lag/lead indicators

Moderation

  • arousal held in optimal zone
  • parasympathetic rebound within hours
  • allocate resources via Pareto principle
  • ego depletion detected at performance dip

Self-Reflection

  • daily introspection
  • experiences distilled to if-then rule
  • label current emotion with one word
  • immediate behavioral calibration
